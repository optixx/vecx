vecx
====

![Vectrex Loading Screen](screenshot0.png) ![Star Trek](screenshot1.png)

Requirements
------------
* `libsdl2`
* `sdl2_gfx`
* `sdl2_image`

```brew install  sdl2  sdl2_gfx sdl2_image```

Usage
-----

```vecx game.bin```
